# RGIT CODECELL

RGIT CodeCell's official website built with Next.js and deployed on Vercel.

## Tech Stack & Deployment
>Can access following dashboard by logging in with your Google account (rgitcodecell@gmail.com)
- Deployed on Vercel
- Domain: rgitcodecell.in (Hostinger)
- Images: Hosted on Cloudinary

## Content Management

All website content can be modified by editing the TypeScript files located in:

- `src/lib/teams/`
- `src/lib/events/`
- `src/lib/founders/`
- `src/lib/galleryImages/`
- `src/lib/shortlisted/`

## Contributing

Make sure to create a branch before making any changes, naming convention: `<username>/<feature-you-working-on>`.
Ex: `DhruvLohar/create-recursion-page-2025`
>Merge the branch into `main` only when reviewed and completed the feature.
